# Privacy

Numen v1 collects no analytics.

Drafts, BibTeX, figure specs, and project settings are stored locally in the user's browser through IndexedDB. They are not uploaded by the app.

The frontend fetches public static artifacts from `https://baditaflorin.github.io/numen/data/v1/`.

Links to GitHub and PayPal navigate to external sites with their own privacy policies:

- https://github.com/baditaflorin/numen
- https://www.paypal.com/paypalme/florinbadita
